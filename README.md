# Interactive Robot

## Overview
Interactive Robot is a browser-based project built with JavaScript and HTML5 Canvas.  
It features a minimalist robot that can walk, shoot lasers, spin its arms, and respond to user interactions in real time.  
The design is fully responsive, ensuring smooth performance across devices from desktop to mobile.

## Features
- **Power Button:** Toggles the robot on and off (eyes glow, indicator turns green).  
- **Charge Button:** Recharges the battery (restores 3 actions, 3-second charge).  
- **Canvas Click:** Fires lasers from the robot’s eyes.  
- **Side Click:** Moves the robot toward your pointer.  
- **Arm Click:** Spins the robot’s arms.

## Responsive Design
The layout automatically scales based on the device’s screen size.  
If the display appears misaligned, simply refresh the page.

## Tech Stack
- JavaScript (Vanilla)  
- HTML5 Canvas  
- CSS (Responsive)
