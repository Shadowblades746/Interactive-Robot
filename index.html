Explanation of Software Usage

This software allows you to control a minimalist robot using the buttons and the canvas area. The following actions are implemented:

  • Power Button: Toggles the robot's power on and off. When powered on, the robot's eyes light up and the indicator becomes green
  • Charge Button: Charges the robot's battery, resetting the number of actions to 3. The charging process takes 3 seconds.
  • Canvas Click: Clicking on the canvas in the non-marked areas makes the robot shoot lasers from its eyes.
  • Left/Right Side Click: Clicking on the left or right side of the canvas in the marked areas moves the robot to your pointer
  • Arm Click: Clicking on the robot's arms makes them spin.

The program is designed to run on various devices including smartphones and PCs. It uses responsive design techniques to adjust the canvas size based on the device's screen size, ensuring compatibility across different hardware platforms.
